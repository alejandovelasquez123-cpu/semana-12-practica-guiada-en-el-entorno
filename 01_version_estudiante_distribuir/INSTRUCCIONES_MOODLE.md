# Texto sugerido para Moodle

## Nombre de la actividad

Semana 12 - Práctica guiada en VS Code: scripts, respaldos y reportes

## Descripción

En esta actividad realizarás una práctica guiada en Visual Studio Code. Ejecutarás un proyecto en Python que automatiza tareas repetitivas relacionadas con la gestión de datos de una finca rural. El programa lee información desde un archivo CSV, valida datos, genera reportes, crea respaldos y construye un dashboard visual.

La actividad debe realizarse de manera individual o en parejas, según indique el docente. Debes documentar el proceso mediante una bitácora técnica y capturas de pantalla.

## Evidencia de entrega

Debes entregar:

1. Bitácora completa.
2. Taller práctico respondido.
3. Capturas de pantalla del proceso.
4. Reporte generado.
5. Evidencia del respaldo creado.
6. Captura del dashboard.
7. Conclusión personal.

## Criterios generales

Se evaluará:

- Uso correcto de VS Code y terminal.
- Ejecución del script.
- Identificación de entradas y salidas.
- Análisis de reportes y respaldos.
- Interpretación de errores.
- Calidad de la bitácora.
- Claridad en la conclusión.
