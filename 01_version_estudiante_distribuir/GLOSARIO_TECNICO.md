# Glosario técnico de la práctica

Completa este glosario con tus propias palabras. Puedes usar las definiciones base, pero debes agregar un ejemplo aplicado al proyecto.

| Término | Definición base | Ejemplo aplicado |
|---|---|---|
| Script | Programa corto que automatiza una tarea específica. | Un archivo Python que genera reportes semanales. |
| Automatización | Proceso de hacer que una tarea se ejecute con menor intervención manual. | Calcular el total de leche sin hacerlo a mano. |
| Respaldo | Copia de seguridad de un archivo. | Copia del CSV en `outputs/respaldos`. |
| Reporte | Documento que resume datos y resultados. | Archivo TXT con total y promedio semanal. |
| CSV | Archivo de texto organizado por filas y columnas separadas por comas. | `produccion_finca.csv`. |
| JSON | Formato estructurado para guardar datos. | Resumen semanal con indicadores. |
| Dashboard | Vista visual de información. | Página `index.html` con tarjetas y tabla. |
| Log | Registro técnico de eventos de ejecución. | Archivo que indica si el programa funcionó. |
| Validación | Revisión de datos antes de procesarlos. | Detectar litros negativos. |
| Error | Problema que impide o afecta el funcionamiento correcto. | Una fecha inválida en el CSV. |
| Advertencia | Aviso de posible problema que no siempre detiene el programa. | Producción muy baja en un día. |
| Terminal | Herramienta para ejecutar comandos. | Lugar donde se escribe `python src/main.py`. |
| Ruta | Ubicación de un archivo o carpeta. | `data/produccion_finca.csv`. |
| Entrada | Datos que recibe el programa. | Archivo CSV. |
| Salida | Resultado generado por el programa. | Reporte, respaldo y dashboard. |
| Depuración | Proceso de encontrar y corregir errores. | Revisar por qué un dato no se procesa. |
| Indicador | Resultado calculado que ayuda a analizar datos. | Promedio diario de leche. |
| Evidencia | Prueba de que se realizó una acción. | Captura de pantalla o archivo generado. |
| Bitácora | Documento donde se registra el proceso realizado. | Plantilla completada por el estudiante. |
| Reproducibilidad | Capacidad de repetir un proceso y obtener resultados similares. | Ejecutar el mismo script con el mismo CSV. |
