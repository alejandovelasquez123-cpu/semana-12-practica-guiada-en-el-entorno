# Ejercicio 1 - Explorar el proyecto

## Objetivo

Reconocer la estructura del proyecto antes de ejecutar código.

## Instrucciones

1. Abre el proyecto en VS Code.
2. Identifica las carpetas principales.
3. Abre el archivo `README.md`.
4. Abre el archivo `data/produccion_finca.csv`.
5. Abre la carpeta `src/finca_auto`.

## Preguntas

1. ¿Cuántas carpetas principales tiene el proyecto?
2. ¿Dónde están los datos de entrada?
3. ¿Dónde está el código fuente?
4. ¿Dónde se guardarán los resultados?
5. ¿Qué archivo parece iniciar el programa?

## Evidencia

Toma una captura del explorador de VS Code.
