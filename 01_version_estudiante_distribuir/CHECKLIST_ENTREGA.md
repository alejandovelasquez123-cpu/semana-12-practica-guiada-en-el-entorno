## Archivos sugeridos para entregar

Puedes entregar un solo ZIP con:

```text
bitacora.pdf                          → BITACORA_PLANTILLA.md (convertido a PDF)
capturas/                             → carpeta evidencias/ con 8 imágenes
reporte_generado.txt                  → un reporte de outputs/reportes/
respuesta_taller.pdf                  → TALLER_PRACTICO.md (convertido a PDF)
ejercicio_1_exploracion.pdf           → 01_EXPLORAR_EL_PROYECTO.md
ejercicio_2_ejecucion.pdf             → 02_EJECUTAR_Y_OBSERVAR.md
ejercicio_3_errores.pdf               → 03_ANALIZAR_ERRORES.md
ejercicio_4_datos.pdf                 → 04_MODIFICAR_DATOS.md
ejercicio_5_reto.pdf                  → 05_RETO_DE_MEJORA.md
VALIDACION_COMPLETITUD.md             → Reporte de validación
```

No subas carpetas innecesarias como `__pycache__` o entornos virtuales.
