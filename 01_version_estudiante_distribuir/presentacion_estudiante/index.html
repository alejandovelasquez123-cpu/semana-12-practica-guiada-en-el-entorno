Semana 12: Scripts, respaldos y reportes

Práctica guiada para comprender cómo un script puede leer datos, procesarlos, generar reportes, crear respaldos y presentar resultados visuales.

1. Datos

El archivo CSV contiene registros diarios de producción de una finca.

2. Script

Python ejecuta instrucciones para validar, calcular y generar resultados.

3. Reporte

El reporte resume indicadores útiles para tomar decisiones.

4. Respaldo

El respaldo protege la información original contra pérdidas o cambios accidentales.

Flujo de trabajo

CSV
→
Validación
→
Cálculos
→
Reporte
→
Respaldo
→
Dashboard
Comando principal:
python src/main.py

Pregunta clave

¿Qué datos entran, qué proceso ocurre y qué resultados salen?

Evidencia

Capturas, bitácora, reporte, respaldo y dashboard.

Reto

Modificar datos, analizar errores y proponer una mejora.
